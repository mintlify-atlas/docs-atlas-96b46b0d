The Incel Test categorizes users into four distinct result types based on their **personality** and **looks** scores. Each result type represents a unique combination of these two dimensions.

## Overview

Results are determined by evaluating where your scores fall on two axes:

- **Personality Axis**: Beta (negative) to Chad (positive)
- **Looks Axis**: Beta (negative) to Chad (positive)

The four quadrants create distinct archetypes that reflect different combinations of personality and appearance.

```javascript
    $("#congrats-message").addClass('bg-secondary mb-3');
    $("#congrats-message-header").prop('innerHTML', "You are Jared Fogle.");

```

This result appears when both personality and looks scores are perfectly balanced at zero.

**Location in code**: `questioner.js:138-142`

## Implementation Details

The result determination logic in the `finale()` function follows this structure:

1. Scores are normalized to a range of approximately -253 to +253
2. The personality score is inverted (`-1 * personality`)
3. Conditional statements check which quadrant the scores fall into
4. The appropriate result card is displayed with its color theme and description

<Tip>
The order of conditional checks matters! The function checks conditions in this sequence: zero scores, Beta Beta, Beta Chad, Chad Beta, then Chad Chad.
</Tip>
