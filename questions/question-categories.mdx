The Incel Test uses two distinct categories of questions to assess users across different dimensions: **looks-based questions** and **personality-based questions**. Each question belongs to one of these categories and contributes to a separate score that determines the user's final position on the Chad-Beta spectrum.

## Question Structure

Each question in `questions.json` follows this structure:

```json
{
  "prompt": "Question text",
  "isLooks": true,
  "weight": 1,
  "answers": [
    "Answer option 1",
    "Answer option 2",
    "Answer option 3",
    "Answer option 4",
    "Answer option 5"
  ]
}
```

### Properties

- **prompt**: The question text displayed to the user
- **isLooks**: Boolean flag determining the category (`true` for looks, `false` for personality)
- **weight**: Numerical weight (1-5) indicating the question's importance in scoring
- **answers**: Array of answer options (2-5 choices)

## Weighting System

The weighting system allows certain questions to have more influence on the final score:

- **Weight 1**: Standard questions (most common)
- **Weight 2**: Important questions (e.g., height, athletic build, anime viewing)
- **Weight 3**: Very important questions (e.g., pornography preferences)
- **Weight 5**: Critical questions (e.g., sexual experience)

### How Weights Work

Weights are applied during score calculation in the `calcScore()` function:

```javascript
function calcScore(answer, options) {
    n = options / 2;
    if ((options % 2) != 0) {
        console.log("│  Odd number of answers.")
        if (answer < ((options + 1) / 2)) {
            return calcScore(answer, options + 1);
        } else {
            return calcScore(answer + 1, options + 1);
        }
    }
    console.log("│  Even number of answers.")
    return ((answer - n) / n) * currentWeight;
}
```

The calculated score is multiplied by `currentWeight`, meaning a question with weight 2 has twice the impact of a weight 1 question.

### Weight Distribution

As questions are loaded, the system tracks separate weight totals for each category:

```javascript
if (isLooks) {
    console.log("│  Question " + questionId + " is about looks.");
    looksCount += currentWeight;
} else {
    console.log("│  Question " + questionId + " is about personality.");
    personalityCount += currentWeight;
}
```

These totals (`looksCount` and `personalityCount`) are used to normalize scores in the final calculation.
